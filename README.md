# README

This is demo Rails application with react and material ui

Prerequisite

> Ruby - 3.0.1

> Rails - 7.0.1

> Node - v14+

> Bundler - 2.2.31

<b> Set mysql login details to database.yml </b>

<b> Run </b>

<code>bin/setup</code> # for setup rails project

<code>bin/dev</code> # for run rails application
